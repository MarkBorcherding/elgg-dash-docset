Authenticate against a list of username and passwords.

HTTP Digest Authentication doesn't work with PHP in CGI mode, you have to add this into your .htaccess RewriteRule .* - [E=HTTP_AUTHORIZATION:%{HTTP:Authorization},L]

Parameters
string$realmName of the authentication session
array$usersAn assoc array of username and password: array('uname1'=>'pwd1', 'uname2'=>'pwd2')
string$fail_msgMessage to be displayed if the User cancel the login
string$fail_urlURL to be redirect if the User cancel the login
Returns
string The username if login success.
