Detailed Description

Forms a front controller for a console app, handling and validating arguments (options)

Instantiate, add arguments, then call validate(). Afterwards, the user's valid arguments and their values will be available in $cli->values.

You may also specify that some arguments be used to provide input/output. By communicating solely through the file pointers provided by openInput()/openOutput(), you can make your app more flexible to end users.
