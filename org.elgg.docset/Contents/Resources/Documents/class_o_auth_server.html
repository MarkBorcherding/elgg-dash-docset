Member Function Documentation

accessToken ( )

Exchange a request token for an access token. The exchange is only succesful iff the request token has been authorized.

Never returns, calls exit() when token is exchanged or when error is returned.

authorizeFinish (   $authorized,
  $user_id 
)

Overrule this method when you want to display a nice page when the authorization is finished. This function does not know if the authorization was succesfull, you need to check the token in the database.

Parameters
booleanauthorized if the current token (oauth_token param) is authorized or not
intuser_id user for which the token was authorized (or denied)
authorizeVerify ( )

Verify the start of an authorization request. Verifies if the request token is valid. Next step is the method authorizeFinish()

Nota bene: this stores the current token, consumer key and callback in the _SESSION

Exceptions
OAuthExceptionthrown when not a valid request

array token description
