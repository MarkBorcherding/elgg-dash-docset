Construct the request to be signed. Parses or appends the parameters in the params url. When you supply an params array, then the params should not be urlencoded. When you supply a string, then it is assumed it is of the type application/x-www-form-urlencoded

Parameters
stringrequest url
stringmethod PUT, GET, POST etc.
mixedparams string (for urlencoded data, or array with name/value pairs)
stringbody optional body for PUT and/or POST requests

Parameters

setUnsigned ( )

Reset the 'signed' flag, so that any changes in the parameters force a recalculation of the signature.

Sign our message in the way the server understands. Set the needed oauth_xxxx parameters.

Parameters
intusr_id (optional) user that wants to sign this request
arraysecrets secrets used for signing, when empty then secrets will be fetched from the token registry
stringname name of the token to be used for signing
Exceptions
